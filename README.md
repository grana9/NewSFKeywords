This project focuses on enhancing the QForce library within Copado Robotic Testing (CRT) by introducing reusable custom keywords to address common challenges in Salesforce UI automation. 
These enhancements aim to improve test stability, reduce script complexity, and standardize automation practices across teams.

Please refer to the link below for the usage guide, which illustrates the step-by-step process of using these keywords. 
Additionally, the definitions of the keywords can be found under the Completed section at the following path: tests\salesforceTests.

Link to Usage guide: https://docs.google.com/document/d/1IqOjJHeHtKiefbklN3ZNHTMbwfzIE-jqeHbBNLPOLR0/edit?tab=t.0#heading=h.x916fn854yvu
